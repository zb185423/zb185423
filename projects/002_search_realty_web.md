
不動産検索Webアプリケーション
================

# 工期
  
2か月
# ユーザ
  
賃貸不動産を探すエンドユーザ
# 概要

- Web上で賃貸不動産を検索するサイト
- 立地、賃料、間取りなどを検索条件に指定して検索することができる
- 気になる物件をCookieに保存し、一覧画面で比較検討することができる
